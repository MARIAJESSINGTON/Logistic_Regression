Logistic Regression is a machine learning algorithm used to predict categories, usually two outcomes like Yes/No or 0/1. It is mainly used for classification problems. The model takes input data (such as age, salary, or glucose level) and calculates the probability of a particular class. Based on that probability, it decides the final output. Logistic Regression is commonly used in real-world applications like disease prediction, spam email detection, and fraud detection.
